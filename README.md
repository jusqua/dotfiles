# dotfiles
My dotfiles

## Requirements
- Neovim (Text Editor)
- Alacritty (Terminal Emulator)
- Fish (Terminal Shell)
- Exa (`ls` alternative)
- Bat (`cat` alternative)
- Ripgrep (`grep` alternative)
- Fd (`find` alternative)
- Starship (Shell Prompt)
- Clang (C compiler)

### For fedora
```
sudo dnf install fd-find starship neovim exa bat ripgrep stow clang
```

## Requires GNU Stow for dotfiles management
##### Execute:
```sh
stow .
```
