# dotfiles
My dotfiles

# Requirements
- Neovim (Text Editor)
- Alacritty (Terminal Emulator)
- Fish (Terminal Shell)
- Exa (`ls` alternative)
- Bat (`cat` alternative)
- Ripgrep (`grep` alternative)
- Fd (`find` alternative)
- Starship (Shell Prompt)

# Requires GNU Stow for dotfiles management
##### Execute:
```sh
stow .
```
